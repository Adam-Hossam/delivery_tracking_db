# Delivery Tracking System Database

This project is a package delivery and tracking database system developed using Microsoft Access as part of a university group project.

The system manages different parts of the logistics process including:
- Orders
- Payments
- Shipment tracking
- Warehouses
- Staff and departments
- Vehicles and packages
- Returns and customer data

The database was built using connected tables, forms, reports, and queries to maintain organized and accurate data throughout the system.

## Features

- Order and payment management
- Shipment tracking system
- Warehouse and vehicle records
- Reports for delivery locations and rejected payments
- Queries for analytics and data filtering
- User-friendly forms for data entry
